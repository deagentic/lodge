# Hardware Wireless Tools

**All manual wireless probing scripts have been removed.**

As per the **New Stack Mandate** in `AGENTS.md`, any tools required to interface with NFC, RFID, Bluetooth, or other wireless protocols must be dynamically built by the **Tool Writer Agent**.

When an agent needs to probe a wireless protocol, they must first generate generalizable tools in this directory and iteratively improve them as they run experiments. Do not rely on legacy, hardcoded protocol analyzers.
