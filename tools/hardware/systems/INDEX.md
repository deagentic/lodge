# Hardware Systems Tools

**All manual hardware systems scripts have been removed.**

As per the **New Stack Mandate** in `AGENTS.md`, any tools required to interface with embedded systems, smart cards, or general hardware must be dynamically built by the **Tool Writer Agent**.

When an agent needs to probe a hardware system, they must first generate generalizable interfacing tools in this directory and iteratively improve them as they experiment. Do not rely on legacy, hardcoded hardware probes.
